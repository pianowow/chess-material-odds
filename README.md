# Chess Material Odds

This project provides a Python script to generate a CSV file (`material_odds.csv`) that enumerates different combinations of chess piece handicaps and their corresponding material values. This can be used to create a ladder of material odds for chess variants or training exercises.

The "handicapped" side is the one with less material. The values are based on a common system (e.g., Pawn=1, Knight=3, Bishop=3.5, Rook=5.5, Queen=9.5).

## Generating the Data

The `material_odds.csv` file can be generated by running the Python script. No external libraries are needed.

```bash
python3 generate_material_odds_csv.py
```

A new `material_odds.csv` file will be created in the same directory.

## Data Format

The generated CSV file contains the following columns:

*   `row_id`: A unique identifier for the row, starting from 1.
*   `handicapped_pawns`: The number of pawns on the handicapped side.
*   `handicapped_knights`: The number of knights on the handicapped side.
*   `handicapped_bishops`: The number of bishops on the handicapped side.
*   `handicapped_rooks`: The number of rooks on the handicapped side.
*   `handicapped_queens`: The number of queens on the handicapped side.
*   `handicapped_total_pieces`: The total number of pieces for the given combination.
*   `handicapped_total_value`: The sum of the material values for the pieces, sorted to create a ladder of increasing material advantage.
